#Stranger vibes
stranger vibes is a dark-themed creative website inspired by a sci-fi atmosphere
this project is build using **only html nd css**.
----
#features
dark cinematic UI design
**upside down animation created using pure css**
background image with overlay effect 
custom buttons and hover effects
clean and simple page layouts.
---
## technologies used
HTML
CSS

##Learning Outcomes
writing structured HTML code
styling layouts using CSS
creating animation with css @keyframes
working with background images and overlays
improving UI design skills.
